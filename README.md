# 影视链 - 视频解析播放器

## 📱 项目简介

**影视链** 是一款基于 HarmonyOS Next 开发的视频解析播放器应用。它通过集成多个第三方解析接口，帮助用户突破各大视频网站的限制，实现一站式搜索、解析和播放全网视频内容。

### ✨ 核心功能

- **多平台支持**：腾讯视频、优酷视频、哔哩哔哩、爱奇艺、芒果TV
- **智能视频解析**：自动检测视频页面，一键解析播放
- **搜索记录**：保存搜索历史，快速重复搜索
- **全屏手势**：滑动快进快退、单击暂停/播放、双击退出全屏
- **横屏优化**：自动横屏并缩放页面，获得更好观看体验
- **自定义解析接口**：支持用户自定义和切换解析服务器

## 📖 使用指南

### 基础使用流程

1. **选择视频网站**：点击顶部下拉菜单，选择要搜索的平台
2. **输入关键词**：在搜索框输入视频名称（如“斗破苍穹”）
3. **执行搜索**：点击“搜索”按钮
4. **进入视频页**：在搜索结果中点击目标视频
5. **解析播放**：点击底部“解析播放”按钮，自动加载解析视频

### 全屏手势操作

| 手势 | 功能 |
|------|------|
| 单击屏幕 | 暂停/播放视频 |
| 左右滑动 | 快进/快退（显示时间面板） |
| 双击屏幕 | 退出全屏 |
| 点击全屏按钮 | 进入全屏模式 |

### 解析接口切换

1. 点击顶部设置按钮（齿轮图标）
2. 在下拉菜单中选择解析网站：
   - 虾米解析（默认）
   - 1907解析
   - 8090解析
   - 全民解析
   - 自定义
3. 点击“保存设置”

### 搜索记录管理

- 点击搜索框自动显示最近搜索记录
- 点击记录快速搜索
- 点击“清空”按钮删除所有记录
- 记录自动保存，应用重启后仍存在

## 🔧 配置文件说明

### 解析接口配置

在 `SimpleBrowser.ets` 中可修改预设解析接口：

```typescript
private parserUrls: string[] = [
  'https://jx.xmflv.cc/?url=',      // 虾米解析
  'https://im1907.top/?jx=',        // 1907解析
  'https://www.8090g.cn/jiexi/?url=', // 8090解析
  'https://api.9527zy.com/jiexi/?url=', // 全民解析
  ''  // 自定义
];
```

### 搜索网站配置

```typescript
private searchUrls: string[] = [
  "https://v.qq.com/x/search/?q=",      // 腾讯视频
  "https://so.youku.com/search/q_",     // 优酷视频
  "https://search.bilibili.com/all?keyword=", // B站
  "https://so.iqiyi.com/so/q_",         // 爱奇艺
  "https://so.mgtv.com/so?k="           // 芒果TV
];
```

## 🐛 已知问题

1. **部分网站视频无法解析**：由于视频网站反爬机制，部分视频可能解析失败
2. **进度条不灵敏**：部分解析接口返回的播放器进度条可能响应不灵敏
3. **横屏缩放可能异常**：部分页面在横屏缩放后可能出现布局错乱
